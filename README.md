This repository provides an example of a pipeline for generating crash simulation data using OpenRadioss.The generated datasets are specifically formatted to train AI models within the PhysicsNemo framework, following [**the PhysicsNemo crash example**](https://github.com/NVIDIA/physicsnemo/tree/main/examples/structural_mechanics/crash).<br>
Using the [**OpenRadioss Bumper Beam example**](http://openradioss.atlassian.net/wiki/spaces/OPENRADIOSS/pages/11075585/Bumper+Beam) as a baseline, the raw data was generated by iteratively modifying the shell thickness and executing successive simulation runs.
